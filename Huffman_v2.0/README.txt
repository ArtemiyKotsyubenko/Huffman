Program awaits 3 arguments.
If you want to decode file - source_file, dest_file, "--decode" flag
If you want to encode file - source_file, dest_file, "--encode" flag
Examples:
./arch /home/user/source.txt /home/user/arch.txt --encode
./arch /home/user/arch.txt /home/user/decoded.txt --decode

where arch is compiled file
